# Детали брони

**ID:** SCR-006  
**Тип:** Страница  
**Домен:** 02. Бронирование  
**Приоритет:** Critical  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Страница с полной информацией о брони. Позволяет просмотреть все детали и при необходимости отменить запись.

### User Story

> Как клиент скалодрома, я хочу видеть полную информацию о своей записи и иметь возможность отменить её, если мои планы изменились.

## Элементы экрана

### 1. Бейдж статуса

| Статус | Отображение |
|:---|:---|
| Активна | Зеленый бейдж «Активна» |
| Отменена клиентом | Серый бейдж «Отменена» |
| Поздняя отмена | Оранжевый бейдж «Поздняя отмена» |
| Отменена клубом | Красный бейдж «Отменена клубом» |
| Прошедшая | Серый бейдж «Прошедшая» |

### 2. Информация о брони

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Дата и время | Полная дата и время | `Slot.start_at` |
| Название маршрута | Название | `Route.name` |
| Тип маршрута | Бейдж «Новички»/«Опытные» | `Route.type` |
| Инструктор | Имя инструктора | `Instructor.name` |
| Место встречи | Адрес/ориентир | `Slot.meeting_point` |
| Количество мест | «2 места» | `seats_count` |
| Снаряжение | «1 прокатная · 1 своя» | `rental_count` |
| Цена | Итоговая цена | `price_total` |
| Дата создания | «15 июля 2026, 18:20» | `created_at` |
| Дата отмены | (если есть) | `cancelled_at` |

### 3. Подсказка дедлайна

| Элемент | Описание | Отображение |
|:---|:---|:---|
| Текст | «Бесплатно освободить место можно до ХХ:ХХ» | Только для активной брони |

### 4. Кнопка «Отменить запись»

| Состояние | Отображение | Действие |
|:---|:---|:---|
| Активна, до старта | Активная кнопка | Открыть BS-003 |
| Активна, после старта | Неактивная, «Отмена недоступна» | Нет действия |
| Уже отменена | Скрыта | Нет действия |

## Используемые запросы

### GET /bookings/{bookingId}

**Параметры:**

| Параметр | Тип | Описание |
|:---|:---|:---|
| `bookingId` | UUID | Идентификатор брони |

### POST /bookings/{bookingId}/cancel

**Параметры:**

| Параметр | Тип | Описание |
|:---|:---|:---|
| `bookingId` | UUID | Идентификатор брони |

**Обработка ответа:**

| Код | Сценарий | Действие |
|:---|:---|:---|
| 200 | Успех | Обновить статус |
| 422 | Слот стартовал | Сообщение об ошибке |
| 409 | Уже отменена | Актуализировать статус |

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** клиент открывает детали брони, **Когда** экран загружен, **Тогда** показана вся информация о брони | P0 |
| AC-002 | **Дано** бронь активна, **Когда** клиент кликает «Отменить», **Тогда** открывается BS-003 | P0 |
| AC-003 | **Дано** бронь отменена, **Когда** экран загружен, **Тогда** кнопка «Отменить» скрыта | P0 |
| AC-004 | **Дано** слот стартовал, **Когда** экран загружен, **Тогда** кнопка «Отменить» неактивна | P0 |
