# Карточка слота

**ID:** SCR-003  
**Тип:** Страница  
**Домен:** 01. Расписание  
**Приоритет:** Critical  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Страница с полной информацией о тренировке. Содержит все детали маршрута, инструктора, места встречи и карту. Является промежуточным шагом перед оформлением записи.

### User Story

> Как клиент скалодрома, я хочу видеть полную информацию о тренировке, чтобы принять решение о записи.

## Элементы экрана

### 1. Hero-обложка

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Изображение маршрута | Декоративное изображение | `Route.image` (если есть) |
| Плейсхолдер | Нейтральная заглушка | Если изображения нет |

### 2. Информация о тренировке

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Дата и время | Полная дата и время | `Slot.start_at` |
| Название маршрута | Название | `Route.name` |
| Тип маршрута | Бейдж «Новички»/«Опытные» | `Route.type` |
| Длительность | «~90 мин» | `Route.duration_min` |
| Описание | Текст описания | `Route.description` |
| Инструктор | Имя инструктора | `Instructor.name` |
| Место встречи | Адрес/ориентир | `Slot.meeting_point` |
| Цена за место | «1500 ₽» | `Slot.price` |
| Свободные места | «5 из 8» | `Slot.free_seats` |
| Прокатное снаряжение | «7 комплектов» | `Slot.free_rental_boards` |

### 3. Карта маршрута

| Элемент | Описание | Действие |
|:---|:---|:---|
| Превью карты | Статичное изображение | Клик → открыть BS-004 |
| Кнопка «Открыть карту» | — | Открыть BS-004 |

### 4. Кнопка «Записаться»

| Состояние | Отображение | Действие |
|:---|:---|:---|
| Есть места | Активная кнопка | Переход на SCR-004 |
| Нет мест | Неактивная, «Мест нет» | Нет действия |
| Слот отменен | Скрыта/неактивна, «Отменена» | Нет действия |

## Используемые запросы

### GET /slots/{slotId}

**Параметры:**

| Параметр | Тип | Описание |
|:---|:---|:---|
| `slotId` | UUID | Идентификатор слота |

**Ответ:** Объект `Slot` с вложенными `Route` и `Instructor`

## Состояния экрана

| Состояние | Отображение |
|:---|:---|
| Загрузка | Скелетон-загрузка всех блоков |
| Content (есть места) | Полная информация + активная кнопка |
| Content (нет мест) | Полная информация + неактивная кнопка |
| Content (отменен) | Полная информация + затемнение |
| Error | Error state + кнопка «Обновить» |

---

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** клиент открыл карточку слота, **Когда** экран загружен, **Тогда** показаны все детали тренировки | P0 |
| AC-002 | **Дано** есть места, **Когда** клиент кликает «Записаться», **Тогда** открывается SCR-004 | P0 |
| AC-003 | **Дано** нет мест, **Когда** экран загружен, **Тогда** кнопка «Записаться» неактивна | P0 |
| AC-004 | **Дано** слот отменен, **Когда** экран загружен, **Тогда** отображается пометка «Отменена» | P0 |
| AC-005 | **Дано** клиент кликает по карте, **Когда** карта доступна, **Тогда** открывается BS-004 | P1 |
