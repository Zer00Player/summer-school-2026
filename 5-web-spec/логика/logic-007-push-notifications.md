## Обзор

Логика запрашивает разрешение на push-уведомления в вебе через Web Push API. Запрос показывается только после первой успешной записи — в момент, когда ценность уведомлений очевидна.

### User Story

> Как клиент скалодрома, я хочу получать напоминания о тренировках, чтобы не забывать о них.

### Бизнес-ценность

- Снижение неявок
- Высокая конверсия согласия (запрос после записи)
- Один запрос — без навязывания
