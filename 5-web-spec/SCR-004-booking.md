# Оформление записи

**ID:** SCR-004  
**Тип:** Страница  
**Домен:** 02. Бронирование  
**Приоритет:** Critical  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Страница оформления записи. Позволяет клиенту выбрать количество мест (1-3), вариант снаряжения (своё/прокат) для каждого места и подтвердить запись. Показывает итоговую цену.

### User Story

> Как клиент скалодрома, я хочу выбрать количество мест и вариант снаряжения, чтобы записаться на тренировку и быть уверенным, что снаряжение для меня подготовят.

## Элементы экрана

### 1. Сводка слота (read-only)

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Дата и время | Полная дата и время | `Slot.start_at` |
| Название маршрута | Название | `Route.name` |
| Тип маршрута | Бейдж «Новички»/«Опытные» | `Route.type` |
| Инструктор | Имя инструктора | `Instructor.name` |
| Свободные места | «5 мест» | `Slot.free_seats` |
| Прокат | «7 комплектов» | `Slot.free_rental_boards` |

### 2. Выбор количества мест

| Элемент | Описание | Действие |
|:---|:---|:---|
| Кнопка «−» | Уменьшить количество | `seats_count -= 1` |
| Число | Текущее количество | — |
| Кнопка «+» | Увеличить количество | `seats_count += 1` |
| Подпись | «Можно записать до 3 мест» | — |

**Условия доступности:**
- «−» неактивна при `seats_count = 1`
- «+» неактивна при `seats_count = 3` или `seats_count = free_seats`

### 3. Выбор снаряжения

| Элемент | Описание | Действие |
|:---|:---|:---|
| Переключатель «Своё» | Своё снаряжение | Выбрать для этого места |
| Переключатель «Прокат» | Прокатное снаряжение | Выбрать для этого места |
| Счетчик проката | «Прокатных выбрано: N из M» | Отображение |

### 4. Блок цены

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Строка «Места» | `price × seats_count` | `Slot.price`, `seats_count` |
| Строка «Прокат» | `rental_price × rental_count` | `Slot.rental_price`, `rental_count` |
| Итого | Сумма | Расчет |
| Напоминание об оплате | «Оплата на месте...» | Статичный текст |

### 5. Кнопка «Записаться»

| Состояние | Отображение | Действие |
|:---|:---|:---|
| Валидно | Активная кнопка | POST /bookings |
| Невалидно | Неактивная кнопка | Нет действия |
| Отправка | Индикатор загрузки | Блокировка UI |

## Используемые запросы

### POST /bookings

**Заголовки:**

| Заголовок | Значение |
|:---|:---|
| `Authorization` | `Bearer {token}` |
| `Idempotency-Key` | UUID |

**Тело запроса:**

| Параметр | Тип | Описание |
|:---|:---|:---|
| `slot_id` | UUID | Идентификатор слота |
| `seats_count` | integer | Количество мест (1-3) |
| `rental_count` | integer | Количество прокатных комплектов (0..seats_count) |

**Обработка ответа:**

| Код | Сценарий | Действие |
|:---|:---|:---|
| 201 | Успех | Переход на BS-002 |
| 409 | Нет мест / Нет проката | Сообщение об ошибке |
| 410 | Слот отменен | Сообщение об ошибке |
| 422 | Слот стартовал | Сообщение об ошибке |

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** клиент на странице оформления, **Когда** выбирает количество мест, **Тогда** цена пересчитывается в реальном времени | P0 |
| AC-002 | **Дано** клиент выбрал места и снаряжение, **Когда** кликает «Записаться» и запрос успешен, **Тогда** открывается BS-002 | P0 |
| AC-003 | **Дано** клиент выбрал прокат, **Когда** свободных комплектов нет, **Тогда** опция «Прокат» недоступна | P0 |
| AC-004 | **Дано** ошибка при создании брони, **Когда** клиент получает 409, **Тогда** отображается понятное сообщение | P0 |
