# Список слотов

**ID:** SCR-002  
**Тип:** Страница  
**Домен:** 01. Расписание  
**Приоритет:** Critical  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Страница «Список слотов» — детальное представление расписания с расширенной фильтрацией. Отображает все доступные тренировки в виде карточек с полной информацией.

### User Story

> Как клиент скалодрома, я хочу видеть полный список тренировок с возможностью фильтрации, чтобы найти подходящий вариант и записаться.

## Элементы экрана

### 1. Карточка слота (расширенная)

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Дата и время | Крупно, контрастно | `Slot.start_at` |
| Название маршрута | Название тренировки | `Route.name` |
| Описание маршрута | Краткое описание | `Route.description` |
| Тип маршрута | Бейдж «Новички»/«Опытные» | `Route.type` |
| Инструктор | Имя инструктора | `Instructor.name` |
| Свободные места | «5 мест» / «Мест нет» | `Slot.free_seats` |
| Прокатное снаряжение | Доступность | `Slot.free_rental_boards` |
| Цена | «1500 ₽» | `Slot.price` |
| Длительность | «90 мин» | `Route.duration_min` |
| Карточка целиком | Кликабельная область | Переход на SCR-003 |
| Кнопка «Записаться» | — | Переход на SCR-004 |

## Используемые запросы

### GET /slots (с пагинацией)

| Параметр | Тип | Описание |
|:---|:---|:---|
| `date_from` | string | Начальная дата |
| `date_to` | string | Конечная дата |
| `route_type` | array | Типы маршрута |
| `instructor_id` | array | Инструкторы |
| `only_available` | boolean | Только свободные |
| `limit` | integer | Размер страницы |
| `offset` | integer | Смещение |

### GET /instructors

| Параметр | Тип | Описание |
|:---|:---|:---|
| `limit` | integer | Размер страницы |
| `offset` | integer | Смещение |
