# TC-04: Просмотр своих записей

**Приоритет:** P0  
**Связан с:** US-04, ФТ-07

## Предусловие
- Приложение запущено
- Есть хотя бы одна запись (создана через TC-03 или уже существует)

## Шаги
1. В нижней навигации кликнуть на "Мои записи"
2. Посмотреть на список записей
3. Кликнуть на любую запись

## Ожидаемый результат
- ✅ Открывается экран "Мои записи"
- ✅ Вверху видны вкладки: Все, Активные, Прошедшие
- ✅ Отображается список записей
- ✅ Каждая запись показывает:
  - Дату и время
  - Название тренировки
  - Статус (например, "Активна")
- ✅ При клике на запись открываются детали

## Критерии приёмки
- [ ] Переход на экран записей работает
- [ ] Вкладки переключаются
- [ ] Список записей отображается
- [ ] Детали записи открываются при клике
