# TC-03: Простая запись на тренировку

**Приоритет:** P0  
**Связан с:** US-02, ФТ-04, ФТ-05

## Предусловие
- Приложение запущено
- Находимся на главном экране
- Есть хотя бы одна тренировка со свободными местами (например, "3/8")

## Шаги
1. Найти тренировку с кнопкой "Записаться"
2. Кликнуть на кнопку "Записаться"
3. На экране подтверждения выбрать "Со своим снаряжением"
4. Кликнуть на кнопку "Подтвердить запись"

## Ожидаемый результат
- ✅ Открывается экран подтверждения с деталями тренировки
- ✅ Видны две карточки выбора снаряжения
- ✅ После выбора снаряжения кнопка "Подтвердить запись" становится активной
- ✅ Появляется модальное окно "ВЫ ЗАПИСАНЫ!"
- ✅ В модальном окне видна сводка: дата, время, маршрут, цена
- ✅ Есть кнопки "Мои записи" и "Готово"

## Критерии приёмки
- [ ] Переход к экрану подтверждения работает
- [ ] Выбор снаряжения работает
- [ ] Кнопка активна только после выбора
- [ ] Модальное окно успеха появляется
- [ ] Сводка брони отображается корректно
