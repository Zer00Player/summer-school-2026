# TC-05: Простая отмена записи

**Приоритет:** P1  
**Связан с:** US-04, ФТ-07

## Предусловие
- Приложение запущено
- Есть активная запись (статус "Активна")
- До начала тренировки больше 2 часов

## Шаги
1. Перейти в "Мои записи"
2. Кликнуть на активную запись
3. На экране деталей найти кнопку "Отменить запись"
4. Кликнуть на кнопку "Отменить запись"
5. В модальном окне подтверждения кликнуть "Да, отменить"

## Ожидаемый результат
- ✅ На экране деталей видна кнопка "Отменить запись"
- ✅ Появляется модальное окно с вопросом "Отменить запись?"
- ✅ В модальном окне объясняется правило 2 часов
- ✅ После подтверждения запись отменяется
- ✅ Статус записи меняется на "Отменена"

## Критерии приёмки
- [ ] Кнопка отмены видна для активной записи
- [ ] Модальное окно подтверждения появляется
- [ ] Отмена работает корректно
- [ ] Статус обновляется после отмены
