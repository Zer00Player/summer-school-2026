# TC-02: Просмотр расписания

**Приоритет:** P0  
**Связан с:** US-01, ФТ-01, ФТ-02

## Предусловие
- Приложение запущено (TC-01 пройден)
- Находимся на главном экране "Расписание"

## Шаги
1. Посмотреть на верхнюю часть экрана
2. Найти горизонтальный список дат (чипы)
3. Кликнуть на чип "Завтра" (или любую другую дату)
4. Посмотреть на список тренировок

## Ожидаемый результат
- ✅ Вверху видны даты на 7 дней (Сегодня, Завтра, и т.д.)
- ✅ При клике на другую дату список тренировок меняется
- ✅ Каждая тренировка показывает:
  - Время (например, "18:00")
  - Название (например, "Болдеринг для новичков")
  - Тип (Новичок/Опытный)
  - Количество занятых мест (например, "5/8")
  - Кнопку "Записаться" (если есть места)

## Критерии приёмки
- [ ] Фильтр по датам работает
- [ ] Список тренировок обновляется при смене даты
- [ ] Информация о тренировках отображается корректно
- [ ] Формат "занято/всего" виден (например, "5/8")
