# TC-01: Запуск приложения

**Приоритет:** P0  
**Связан с:** US-01, ФТ-01

## Предусловие
- Файл `index.html` существует
- Браузер (Chrome/Firefox/Safari) установлен

## Шаги
1. Открыть папку с проектом
2. Дважды кликнуть на файл `index.html`

## Ожидаемый результат
- ✅ Приложение открывается в браузере
- ✅ Видна шапка с заголовком "🧗 Скалодром"
- ✅ Отображается список тренировок на сегодня
- ✅ Внизу видна навигация: Расписание, Мои записи, Уведомления

## Критерии приёмки
- [ ] Приложение запускается без ошибок
- [ ] Основной экран отображается корректно
- [ ] Навигация видна и кликабельна
