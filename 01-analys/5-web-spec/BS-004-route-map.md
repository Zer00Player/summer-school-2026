# Карта маршрута

**ID:** BS-004  
**Тип:** Модальное окно  
**Домен:** 01. Расписание  
**Приоритет:** Medium  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Модальное окно с интерактивной картой маршрута. Показывает линию маршрута, место встречи и позволяет открыть карту в Яндекс.Картах.

### User Story

> Как клиент скалодрома, я хочу посмотреть маршрут на интерактивной карте, чтобы понять, как проходит тренировка и где находится место встречи.

## Элементы экрана

### 1. Интерактивная карта

| Элемент | Описание | Технология |
|:---|:---|:---|
| Полотно карты | Интерактивная карта | Yandex Maps API |
| Линия маршрута | Полилиния маршрута | `Route.geometry` |
| Пин места встречи | Метка точки сбора | `meeting_point_lat/lng` |
| Контролы зума | ＋/− | Yandex Maps API |

### 2. Текстовый блок

| Элемент | Описание | Источник данных |
|:---|:---|:---|
| Название и тип | «Городская набережная · Новичковый» | `Route.name`, `Route.type` |
| Длительность | «~90 мин» | `Route.duration_min` |
| Место встречи | Адрес/ориентир | `Slot.meeting_point` |

### 3. Кнопки действий

| Элемент | Описание | Действие |
|:---|:---|:---|
| «Проложить маршрут» | Навигация до места | Открыть Яндекс.Карты с маршрутом |
| «Открыть в Яндекс.Картах» | Просмотр на карте | Открыть Яндекс.Карты с точкой |

| Действие | URL |
|:---|:---|
| Открыть точку | `https://yandex.ru/maps/?pt={lat},{lng}` |

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** клиент открывает карту, **Когда** есть геометрия маршрута, **Тогда** отображается линия маршрута и пин | P0 |
| AC-002 | **Дано** клиент открывает карту, **Когда** клик «Проложить маршрут», **Тогда** открывается Яндекс.Карты | P0 |
| AC-003 | **Дано** клиент открывает карту, **Когда** клик «Открыть в Яндекс.Картах», **Тогда** открывается Яндекс.Карты с точкой | P0 |
| AC-004 | **Дано** ошибка загрузки карты, **Когда** открывается модальное окно, **Тогда** отображается текстовый фолбэк | P0 |
