# Фильтры

**ID:** BS-001  
**Тип:** Модальное окно  
**Домен:** 01. Расписание  
**Приоритет:** High  
**Статус:** Черновик  
**Платформа:** Web (Desktop + Mobile)

---

## Обзор

Модальное окно для фильтрации расписания. Позволяет уточнить параметры поиска тренировок.

### User Story

> Как клиент скалодрома, я хочу фильтровать расписание по датам, типу тренировки и инструктору, чтобы быстро находить подходящие варианты.

## Элементы экрана

### 1. Дата старта

| Элемент | Описание | Действие |
|:---|:---|:---|
| Чип «Сегодня» | Пресет текущего дня | Установить диапазон сегодня |
| Чип «Эта неделя» | Пресет недели | Установить диапазон недели |
| Чип «Выходные» | Пресет выходных | Установить диапазон выходных |
| Поле «С» | Начало диапазона | Выбор даты |
| Поле «По» | Конец диапазона | Выбор даты |

### 2. Тип маршрута

| Элемент | Описание | Действие |
|:---|:---|:---|
| Чип «Новичковый» | Фильтр по novice | Тоггл выбора |
| Чип «Опытный» | Фильтр по experienced | Тоггл выбора |

### 3. Только со свободными местами

| Элемент | Описание | Действие |
|:---|:---|:---|
| Чекбокс | Показать только свободные | Тоггл `only_available` |

### 4. Инструктор

| Элемент | Описание | Действие |
|:---|:---|:---|
| Чип инструктора | Фильтр по инструктору | Тоггл выбора |

### 5. Кнопки

| Элемент | Описание | Действие |
|:---|:---|:---|
| «Сбросить» | Сброс всех фильтров | Вернуть к дефолту |
| «Применить» | Применить фильтры | Закрыть + обновить список |

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** клиент открывает фильтры, **Когда** выбирает фильтры и нажимает «Применить», **Тогда** список обновляется | P0 |
| AC-002 | **Дано** клиент открывает фильтры, **Когда** нажимает «Сбросить», **Тогда** все фильтры сбрасываются | P0 |
| AC-003 | **Дано** клиент открывает фильтры, **Когда** закрывает модальное окно, **Тогда** фильтры не применяются | P0 |
