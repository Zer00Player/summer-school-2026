## Обзор

Логика фильтрации списка слотов:

- **Внутри группы с мультивыбором — OR** (тип маршрута, инструктор)
- **Между группами — AND** (все условия должны выполняться)

По умолчанию показываются слоты на ближайшие 7 дней.

### User Story

> Как клиент скалодрома, я хочу фильтровать расписание по датам, типу тренировки и инструктору, чтобы быстро находить подходящие варианты.

### Бизнес-ценность

- Быстрый поиск подходящих тренировок
- Гибкая комбинация условий
- Предсказуемая выдача
