## Обзор

Логика управляет отменой брони по правилу 2 часов:

- **Ранняя отмена** (≥ 2ч до старта): статус → `cancelled`, места и снаряжение возвращаются
- **Поздняя отмена** (< 2ч до старта): статус → `late_cancel`, места НЕ возвращаются, штрафов нет

Граница ровно 2 часа = ранняя отмена.

### User Story

> Как клиент скалодрома, я хочу отменить запись до старта тренировки, понимая последствия, чтобы освободить место, если планы изменились.

### Бизнес-ценность

- Ранняя отмена возвращает места в слот
- Поздняя отмена честно фиксируется без штрафов
- Единое серверное правило
