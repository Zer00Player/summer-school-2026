## Обзор

Страница «Расписание» — главная страница для авторизованных пользователей. Отображает список доступных тренировок на ближайшие 7 дней с возможностью фильтрации и перехода к записи.

**Ключевые отличия от мобильной версии:**

| Аспект | Мобильное приложение | Веб-приложение |
|:---|:---|:---|
| Отображение | Вертикальный список | Сетка карточек (2-4 колонки) |
| Фильтры | Bottom Sheet | Боковая панель / выпадающие списки |
| Навигация | Тап | Клик + клавиатура |
| Обновление | Pull-to-refresh | Кнопка «Обновить» |

### User Story

> Как клиент скалодрома, я хочу видеть расписание тренировок на неделю вперед, чтобы быстро планировать свои посещения и записываться на занятия.

### Бизнес-ценность

- Самообслуживание записи — снижение нагрузки на администратора
- Визуализация доступности мест в реальном времени
- Удобный просмотр на большом экране

## Используемые запросы

### GET /slots

**Триггер:** Инициализация, изменение фильтров, обновление

**Параметры:**

| Параметр | Тип | Обязательность | Источник |
|:---|:---|:---|:---|
| `date_from` | string | Нет | Состояние |
| `date_to` | string | Нет | Состояние |
| `route_type` | array | Нет | Состояние |
| `instructor_id` | array | Нет | Состояние |
| `only_available` | boolean | Нет | Состояние |
| `limit` | integer | Нет | Константа (20) |
| `offset` | integer | Нет | Состояние |

**Обработка ответа:**

| Результат | UI-реакция |
|:---|:---|
| Загрузка | Скелетон-загрузка карточек |
| Успех (есть данные) | Отображение сетки карточек |
| Успех (нет данных) | Empty state |
| Ошибка 401 | Переход на вход |
| Ошибка 4xx/5xx | Error state + кнопка «Обновить» |
| Ошибка сети | Error state + кнопка «Обновить» |

## Критерии приёмки

| ID | Критерий | Приоритет |
|:---|:---|:---|
| AC-001 | **Дано** пользователь авторизован, **Когда** открывает страницу расписания, **Тогда** отображается сетка тренировок на 7 дней | P0 |
| AC-002 | **Дано** открыто расписание, **Когда** клик «Записаться», **Тогда** открывается страница оформления записи | P0 |
| AC-003 | **Дано** открыто расписание, **Когда** меняется фильтр, **Тогда** список обновляется | P1 |
| AC-004 | **Дано** открыто расписание, **Когда** клик по карточке, **Тогда** открываются детали слота | P1 |
| AC-005 | **Дано** ошибка сети, **Когда** открытие страницы, **Тогда** отображается error state с кнопкой «Обновить» | P0 |
| AC-006 | **Дано** слот отменен, **Когда** отображение карточки, **Тогда** карточка затемнена, кнопка «Записаться» скрыта | P0 |
| AC-007 | **Дано** нет свободных мест, **Когда** отображение карточки, **Тогда** кнопка «Записаться» неактивна | P0 |
