Веб-приложение для самостоятельной записи на тренировки в скалодроме. Заменяет ручную запись через WhatsApp и бумажную тетрадь, устраняя двойные брони и путаницу с местами.

Приложение реализует только роль "Клиент" и включает:
- Просмотр расписания тренировок на 7 дней
- Фильтрация по типу маршрута и инструктору
- Запись на тренировку с выбором снаряжения
- Отмена записи с учётом правила 2 часов
- Бонусная система лояльности
- Оценка инструктора после тренировки
- Система уведомлений

### Дополнительные инструменты
- **Git** — система контроля версий
- **VS Code** — основная IDE для разработки
- **Chrome DevTools** — отладка и тестирование

## Как запустить

### Вариант 1: Через Live Server (рекомендуется)
1. Установите расширение **Live Server** в VS Code
2. Откройте папку `client/` в VS Code
3. Кликните правой кнопкой на `index.html`
4. Выберите **"Open with Live Server"**
5. Браузер автоматически откроется по адресу `http://127.0.0.1:5500/index.html`

### Вариант 2: Простое открытие
1. Откройте папку `client/` в файловом менеджере
2. Дважды кликните на `index.html`
3. Файл откроется в браузере по умолчанию

## Реализованные фичи

### Основные функции
- ✅ **Расписание на 7 дней** с горизонтальным скроллом по датам
- ✅ **Фильтры** по типу маршрута (новичковый/опытный) и инструктору
- ✅ **Запись на тренировку** с выбором снаряжения (со своим/прокат)
- ✅ **Бонусная система** (0.5% от суммы = 1 балл, 1 балл = 1% скидки, макс 50%)
- ✅ **Отмена записи** с правилом 2 часов (ранняя/поздняя отмена)
- ✅ **Оценка инструктора** (1-5 звёзд + комментарий)
- ✅ **Уведомления** о записи, отмене, оценке

### Визуальные состояния
- ✅ **Отменённые клубом слоты** (оранжевый бордер, жёлтый фон, причина отмены)
- ✅ **Полностью заполненные слоты** (серый бордер, полупрозрачность, "Мест нет")
- ✅ **Формат "занято/всего"** (например, 7/8 вместо 1/8)
- ✅ **Скрытие прошедших слотов** для сегодняшнего дня
- ✅ **Empty state** при отсутствии тренировок

### Защита от ошибок
- ✅ **Защита от двойной брони** (нельзя записаться на один слот дважды)
- ✅ **Блокировка бонусов** при балансе ≤ 0
- ✅ **Валидация** перед записью и отменой
